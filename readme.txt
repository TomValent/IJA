     xvalen27 Tomáš Valent

In this state our application just parse file, saves file and shows future layout of our GUI.
Application is still in progress.

How to launch application:
1. open terminal in lib file and run:
	chmod +x get-libs.sh 
	./get-libs.sh
 This will download libraries.
2. Open terminal in xbella01 and run:
	ant compile
 This will compile project.
3. Then run:
	ant run 
 This will run application. 
4. Documentation:
	ant doc
 This will generate documentation.
